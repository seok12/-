<img width="303" alt="스크린샷 2023-06-11 205830" src="https://github.com/seok12/gaekchae/assets/48661266/5735d12e-38dd-428e-9005-af4d4c8d0671">

1. 버튼 클릭시 panel.clear 로 있는 내용을 지운다 
2. panel.add(전환하고싶은 창)으로 다른 창을 삽입하여 화면 전환처럼 보이게 한다
3. 버튼 클릭시 this.dispose로 현재 작업중인 창을 지운다.
